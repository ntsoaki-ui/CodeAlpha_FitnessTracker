# Fitness Tracker App

This is a Java-based Fitness Tracker application developed as part of the CodeAlpha Internship.

## Features
- Add fitness activities
- View activity history
- Delete activities
- SQLite database integration

## Technologies Used
- Java
- SQLite
- IntelliJ IDEA

## How to Run
1. Open the project in IntelliJ IDEA
2. Add sqlite-jdbc-3.43.0.0.jar to libraries
3. Run FitnessTracker.java
